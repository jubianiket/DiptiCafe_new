# **App Name**: Dipti's Orders

## Core Features:

- User Authentication: Secure user login with Admin and Staff roles.
- Order Creation: Create new orders with table number, customer name, items, quantity, and auto-calculated total bill.
- Order Status Tracking: Track order status (Pending, Delivered, Paid) with visual indicators.
- Order Actions: Buttons to mark orders as Delivered or Paid.
- Dashboard: View all active orders and filter by status.
- Sales Summary: Display daily sales summary: total orders and total revenue.
- Admin Panel: Admin view to manage all orders (view, delete).

## Style Guidelines:

- Primary color: Warm amber (#FFB300) to evoke the cozy cafe atmosphere.
- Background color: Light beige (#F5F5DC) for a clean and inviting feel.
- Accent color: Muted brown (#A67B5B) for highlighting important actions and elements.
- Body and headline font: 'PT Sans', a humanist sans-serif that is modern, slightly warm, and suitable for all text needs.
- Simple and clear icons for order statuses and actions.
- Mobile-friendly responsive layout for waiters on the go.
- Subtle transitions for order status changes.
